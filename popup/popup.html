How does this work?

Firefox 149 ships adblock-rust (Brave's Rust engine, MPL-2.0) disabled with no UI, controlled by two about:config prefs.

Standard WebExtensions cannot write arbitrary about:config prefs — so this extension guides you to set them manually once, and manages your filter list so you only copy-paste a single value.

ETP is controlled via the standard browser.privacy WebExtension API and works instantly with no manual steps.
